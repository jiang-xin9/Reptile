# 爬虫案例

#### 介绍
实践各种爬虫案例

#### 软件架构
无特殊架构，详情见提交说明


#### 安装教程

1.  Pycharm、vscode、idea都可以。本系列示例都是Python3.9.0版本

#### 所需要使用到的库

1.  pip install beautifulsoup4
2.  pip install lxml
3.  pip install requests
4.  pip install selenium

如果下载慢则：pip install 模块包名 -i https://pypi.tuna.tsinghua.edu.cn/simple
